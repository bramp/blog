How many ways are there to say phone number?

2013-02-16

In the various systems I’ve worked on, I have seen far too many terms to describe a phone number. I thought I’d catalogue them!

  • CTN – Customer Telephone Number
  • PTN – Personal Telephone Number
  • MTN – Mobile Telephone Number
  • number
  • phone
  • device / device_id
  • subscr_num – subscriber number
  • source_addr / dest_addr – As used by the SMPP spec
  • MSISDN – Mobile Subscriber ISDN Number / Mobile Station International ISDN Number(s) / Mobile International ISDN Number
  • MDN – Mobile Directory Number

any more?
