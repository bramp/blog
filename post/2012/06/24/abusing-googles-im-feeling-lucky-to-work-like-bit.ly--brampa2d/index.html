Abusing Google’s “I’m Feeling Lucky” to work like bit.ly – brampA2D

2012-06-24

I wondered if I could use Google’s I’m Feeling Lucky button just like bit.ly. I would pick a unique word and get it ranked #1 on Google. So I’m going to run a little experiment. The link http://www.google.com/search?q=brampA2D&btnI will do a search on Google and redirect to the #1 link. I’m picking the unique word brampA2D as it does not appear anywhere else online. Hopefully Google’s web crawlers will discover this post and the I’m Feeling Lucky link will start to automatically redirect here.

I imagine spammers could use this to mask their sites behind valid google.com links.
